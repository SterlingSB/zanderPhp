zandarPhp Library
Developed and maintained by Sterling Butts and Andrew Butts.

Overview
A PHP version of the Zandar Library utility suite designed for common general-purpose tasks. It has been developed since 2019 to ensure stability and performance across different environments.

Color Dome
A unique feature of this library is Color Dome, a way of mixing and thinking about color that differs from traditional HSL and HSV models.

A web page where you can try out the mixer is at
